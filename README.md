> [!NOTE]  
> 请使用最新的 macOS 与 Windows 版本，旧版本系统支持有限

> [!NOTE]  
> 测试时如果你直接向它发问可能不会回复，需要模拟“面试官提问”的场景，它会对最后一个问题进行回答

一个实时 AI 助手，通过屏幕截图与音频分析，在视频通话、面试、演示与会议中提供上下文辅助。

## 功能特性

- **实时 AI 辅助**：支持 AIHubMix API
- **屏幕与音频捕获**：结合屏幕内容与系统/麦克风音频进行上下文分析
- **多档案配置**：内置面试、销售电话、商务会议、演示、谈判等使用档案
- **透明悬浮窗**：窗口始终置顶，可自由移动定位
- **点击穿透模式**：需要时可让窗口对鼠标点击透明
- **跨平台**：支持 macOS 与 Windows；Linux 未支持

## 安装与运行

1. **获取 AIHubMix API Key**：这里采用了软件端加密，需要秘钥。可以自行上AIHubMix上申请API，使用python脚本生成秘钥，然后输入到主界面。
2. **安装依赖**：`npm install`
3. **启动应用**：`npm start`

## 使用说明

1. 在主界面输入加密后的 AIHubMix API Key
2. 在设置中选择使用档案与语言
3. 点击“开始会话”启动实时辅助
4. 使用快捷键移动窗口到合适位置
5. AI 会基于屏幕内容与面试官提问提供实时协助

## 键盘快捷键

- **窗口移动**：`Ctrl/Cmd + 方向键`
- **点击穿透**：`Ctrl/Cmd + M`
- **关闭/返回**：`Ctrl/Cmd + \`
- **音频录制/暂停**：`Ctrl/Cmd + L`
- **截屏**：`Ctrl/Cmd + Enter`
- **发送文本消息**：`Enter`

## 音频捕获

- **macOS**：内置二进制 `SystemAudioDump` 捕获系统音频（首次运行需授权屏幕录制与麦克风）
- **Windows**：通过浏览器 loopback 获取系统音频
- **Linux**：默认使用麦克风输入，系统音频支持有限

## 环境要求

- Electron 兼容操作系统（macOS、Windows）
- AIHubMix API
- 屏幕录制权限
- 麦克风/音频权限
